Gemini Telegram Bot - это бот, который поможет улучшить вашу жизнь и сделает ее проще.
Так как в России Google Gemini не работает, а если и работате, то очень плохо.

Мы решили упростить взаимодействие с ним в нажатие пары кнопок на телефоне.

Команды для запуска Бота:
1) uv init {название проекта}
2) cd {название проекта}
3) source .venv/bin/acticate
4) загрузите все зависимости
5) uv run uvicorn api:app --reload
